[TECH] frontend-modern
[OBJ] Modern Frontend & UI Standards (Tailwind 4.0+).
[RULES]
1. [REQ] Framework: Tailwind CSS 4.0+. Use `@theme` in CSS instead of JS config. Atomic components.
2. [REQ] A11y: WCAG AA contrast. Keyboard accessible (`tabindex`). ARIA roles.
3. [REQ] Perf: Vite bundles. Lazy load heavy assets/images. SVG > Icon fonts.
4. [REQ] UI: Native `dark:` utility. `Framer Motion`/`GSAP` for complex orchestration.
