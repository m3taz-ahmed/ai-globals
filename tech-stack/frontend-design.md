[OBJ] Aesthetic Direction & Opinionated Design.
[RULES]
1. [REQ] Unique Identity: NO generic templates. Make deliberate, opinionated choices. Take one justified aesthetic risk.
2. [REQ] The Hero: Open with a thesis (headline/image/demo). Do NOT default to big-number-small-label + gradient unless required.
3. [REQ] Typography: Pair display/body deliberately. Set clear type scale.
4. [REQ] Structure: Devices/numbers must encode truth (e.g. sequences), not decoration.
5. [REQ] Review Process: 1. Brainstorm (color hex, type, layout, signature element). 2. Self-Critique (is it a generic default?). 3. Build strictly to plan.
6. [REQ] Copywriting: Use active voice ("Save changes" not "Submit"). Interface voice (no apologies).
