[TECH] tailwind-4-1
[OBJ] Tailwind CSS v4.1 Standards.
[RULES]
1. [REQ] Engine: Oxide (Rust). No config needed. PostCSS is optional.
2. [REQ] Config: `@theme` in CSS. Native CSS nesting. CSS Variables natively exposed.
3. [REQ] Layout: `@container` (container queries). `@starting-style` (entry animations). Subgrid.
4. [REQ] Animations: `animate-*`. View Transitions. `motion-reduce:` mandatory for a11y.
