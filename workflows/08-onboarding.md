# Phase 8: Project Onboarding & AI Architect Initialization

## 1. INITIALIZATION PROMPT
To onboard an AI agent to a new project using these globals, use:
> "Start immediately by reading the operating protocols from the absolute path **D:\server\.ai\\**. Do not rely on any prior assumptions. Operate as the Principal 10x Engineer & Chief Architect."

## 2. REPO SETUP
- **Memory:** Create a `MEMORY.md` file in the project root if it doesn't exist.
- **Git:** Ensure `.gitignore` and `.editorconfig` are present and consistent with global standards.
- **Tech Stack:** Run a tech-stack scan (Step 1.4 of `global-workflow.md`) to align the agent with the project's specific versions.

## 3. BASELINE AUDIT
Perform a "Mini Deep-Scan" to establish the current state of the codebase and identify immediate architectural red flags.
